Broadcast Messages Check Window

There are four tabs in the Broadcast check window: Release soon, Info awaiting, Released and Info released.

Released soon:

Check all info to be released. If you want your info to be broadcast ahead of time, you may click "Urgent 5CPs" button or "Urgent" button to mark your message urgent.

Urgent 5CPs: Make your meaage go forward five in the messages to be released.

Urgent: Make your message rank 1st in the messages to be released.

Info awaiting:

      Check my messages to be released.

Released:
      Check all the messages that have been broadcast.

Info released:

      Check my messages that have been broadcast.

Note:
      You can not use the feature of Broadcast until you reach level 50.
