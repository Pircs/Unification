Login Interface

1. New account:

(1) Select a group and the server you like.
(2) Type in your account name and password.
(3) Click on Enter button to name your character, Select your job and create your character.

2. Old account:

(1) Select the server Group you are in.
(2) Select the server you are in.
(3) Type in your account name and password.
(4) Click on Enter button to log in the game.
