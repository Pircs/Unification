Set Hotkeys

After you set a hotkey, you may press this key to perform the corresponding command.
                                       

(1) Set a hotkey:

Click on Setup to select Hotkey, select the target interface from Menu, then the target command from Selection, press Ctrl/Shift/Alt + target key, the hotkey name will be shown on the new hotkey box, click on Change, then the hotkey name will be shown on current hotkey box. Then you can press the hotkey to perform the corresponding command.

(2) Restore a hotkey:

Select the target interface from Menu and select the target command from Selection, and then click on Restore.
